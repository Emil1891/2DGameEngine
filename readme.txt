Note that the game engine's classes are grouped together with the game's in the include and src folders. 

The project requires SDL and its add ons and is included as following in the .h and .cpp files: 
#include <SDL2/SDL.h>
#include <SDL2/SDL_mixer.h>
#include <SDL2/SDL_ttf.h>
#include <SDL2/SDL_image.h>
